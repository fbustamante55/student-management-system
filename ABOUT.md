# Acerca del Proyecto

## 📚 Sistema de Gestión de Estudiantes

### Información General

Este proyecto es una aplicación Java desarrollada como práctica académica que implementa un sistema básico de gestión de estudiantes. El sistema permite realizar operaciones fundamentales de administración de datos (CRUD) sobre un registro de estudiantes.

### 🎓 Propósito

El objetivo de este proyecto es demostrar la implementación de:
- Programación orientada a objetos en Java
- Manejo de estructuras de datos (arrays)
- Interfaz gráfica básica con JOptionPane
- Operaciones CRUD (Crear, Leer, Actualizar, Eliminar)
- Gestión de información de estudiantes

### 🔧 Tecnologías Utilizadas

- **Lenguaje**: Java
- **IDE**: NetBeans
- **Interfaz Gráfica**: Swing (JOptionPane)
- **Paradigma**: Programación Orientada a Objetos

### 📋 Funcionalidades Implementadas

1. **Gestión de Estudiantes**
   - Registro de nuevos estudiantes
   - Consulta de estudiantes registrados
   - Modificación de datos de estudiantes
   - Eliminación de estudiantes del registro

2. **Validaciones**
   - Control de capacidad máxima del sistema
   - Búsqueda por cédula para operaciones de modificación y eliminación

### 🏗️ Arquitectura

El proyecto sigue una arquitectura de tres capas:

- **Capa de Modelo**: Clase `Estudiante` que representa la entidad estudiante
- **Capa de Lógica**: Clase `Gestion` que maneja las operaciones de negocio
- **Capa de Presentación**: Clase `Principal` que gestiona la interfaz de usuario

### 📊 Estructura de Datos

El sistema utiliza un array estático de objetos `Estudiante` con una capacidad máxima configurable. Actualmente configurado para 10 estudiantes.
